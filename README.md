Chp
===============

Hard problem solver.


Requirements
----

* [Boost] - the highly comprehensive portable free C++ libraries set
* [g++] - a c++ compiler


Installation
--------------

```sh
git clone github.com/saintfrank/chp
cd chp
make
```

Usage
--------------

```sh
./chp <file_path>
```


[boost]:http://www.boost.org/users/download/
[g++]:http://www.boost.org/users/download/
